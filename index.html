Политика конфиденциальности

Настоящий документ описывает, какие данные обрабатываются на платформе TheVuntgram и каким образом обеспечивается их защита.

1Общие положения

TheVuntgram — закрытая социальная платформа, созданная для общения учащихся общеобразовательных школ, гимназий, лицеев и других учебных заведений. Платформа разрабатывалась с приоритетом на конфиденциальность и безопасность данных несовершеннолетних и совершеннолетних пользователей из числа учащихся. Используя платформу, пользователь также принимает условия Пользовательского соглашения TheVuntgram.

2Какие данные мы не собираем

При регистрации на платформе TheVuntgram не запрашиваются и не хранятся следующие персональные данные:

Адрес электронной почты Номер телефона

Регистрация построена таким образом, чтобы свести объём личных контактных данных, попадающих на сервер, к минимуму.

3Какие данные обрабатываются

При использовании платформы обрабатываются следующие категории данных:

  • Виртуальные данные, автоматически формируемые системой при регистрации аккаунта (внутренние идентификаторы, служебные метки профиля);
  • Сообщения, которыми пользователь обменивается в рамках платформы;
  • Прочие данные о взаимодействии пользователя с платформой (активность внутри приложения, настройки профиля и т. п.).
Важно: ни виртуальные данные, автоматически создаваемые при регистрации, ни сообщения пользователей, ни любые другие данные о пользователе не передаются третьим лицам ни при каких обстоятельствах.

4Меры защиты данных

Для обеспечения безопасности пользовательских данных на платформе применяются следующие меры:

🔒 Шифрование данных

Данные пользователей защищены шифрованием как при хранении, так и при передаче внутри системы.

🛡️ Регулярные проверки безопасности

Платформа регулярно проходит проверки безопасности для своевременного выявления и устранения уязвимостей.

5Кто имеет доступ к данным

Доступ к данным пользователей ограничен исключительно техническими процессами, необходимыми для работы платформы. Передача данных сторонним компаниям, рекламным сетям, аналитическим сервисам или иным третьим лицам не осуществляется.

6Изменения политики

Поскольку проект TheVuntgram находится на стадии разработки и не зарегистрирован официально, данная политика конфиденциальности может дополняться и уточняться по мере развития платформы. Актуальная версия документа всегда доступна пользователям внутри приложения.

7Контакты

По вопросам, связанным с обработкой персональных данных на платформе TheVuntgram, вы можете обратиться к администрации проекта через раздел поддержки внутри приложения.
